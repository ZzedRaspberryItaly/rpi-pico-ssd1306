# SSD1306 OLED with Raspberry Pi Pico
SSD1306 OLED display library for Raspberry Pi Pico using MicroPython. 

The wiring between the Pico and SSD1306 OLED is given below:

| Pico | SSD1306 |
| --- | --- |
| 3V3 (Pin 36) | VDD |
| GND (Pin 38) | GND | 
| I2C1_SDA (GP26) | SDA |
| I2C1_SCL (GP27) | SCK |
